# 📦 Projeto Node.js + TypeScript + MySQL

Este é um projeto backend desenvolvido com Node.js, utilizando TypeScript e MySQL.  
O objetivo é construir uma aplicação web com operações básicas de cadastro, seguindo uma abordagem prática baseada em aulas.

## 🚀 Tecnologias utilizadas

- Node.js
- TypeScript
- Express
- MySQL (mysql2)
- Handlebars
- Bootstrap
- express-fileupload

## 📁 Estrutura do projeto

O projeto segue uma estrutura simples, podendo evoluir ao longo do desenvolvimento:

/src 
├── app.ts 
/views 
/public 

*(Essa estrutura pode ser refatorada conforme o projeto evolui)*

## ⚙️ Como executar o projeto

1. Clone o repositório:
git clone <url-do-repo>

2. Instale as dependências:
npm install

3. Configure o banco de dados:
- Crie um banco MySQL
- Ajuste as credenciais no projeto (idealmente via `.env`)

4. Execute o projeto:
npm run dev
